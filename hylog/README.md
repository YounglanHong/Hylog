## Installation

Use [npm](https://docs.npmjs.com/) to install packages

```js
$ npm install

// 개발 환경 실행
$ npm run dev

// 빌드
$ npm run build
```

## Frontmatter

Go to `_posts` directory

```js
---
title: Post Title
slug: post-slug
date: "00-00-2021"
---
```
